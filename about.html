О чем всё это

Перед Сном ожидайте приятных перемен. Днём ожидайте приятных перемен. После Обеда ожидайте неожиданного праздника.

Перед Сном предостерегайтесь гостей из забытого прошлого. Вечером предостерегайтесь неожиданного праздника. Днём будьте открыты для приятных перемен.

После Обеда предостерегайтесь гостей из забытого прошлого. Ночью ожидайте приятных перемен. Ночью будьте открыты для приятных перемен.

Вечером ожидайте гостей из забытого прошлого. После Обеда будьте открыты для гостей из забытого прошлого. Утром предостерегайтесь неожиданного праздника.

Вечером будьте открыты для встреч со старыми знакомыми. Ночью ожидайте встреч со старыми знакомыми. После Обеда ожидайте приятных перемен.
